Data Cleaning Portfolio

Overview

Welcome to my Data Cleaning portfolio.

This repository contains Python projects demonstrating practical data cleaning techniques using real-world datasets. Each project focuses on improving data quality, documenting the cleaning process, and generating professional PDF reports.

The projects are built using Python and are intended to showcase my skills in data preparation, validation, and automation.

Repository Structure

HR-Cleaning

A complete data cleaning workflow for an HR Attrition dataset.

Features

- Duplicate removal
- Missing value handling
- Text standardization
- Invalid value correction
- Outlier investigation
- Automated PDF report generation

Retail-Cleaning

Features

Laptop-Cleaning

A data cleaning project for an eBay laptop dataset.

Features

Project Goals

The purpose of this repository is to demonstrate practical data cleaning skills, including:

- Handling missing values
- Removing duplicate records
- Standardizing inconsistent data
- Correcting invalid values
- Investigating outliers
- Preparing datasets for analysis
- Automating PDF report generation
